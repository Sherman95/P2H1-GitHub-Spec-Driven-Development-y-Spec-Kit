# UX Spec - Pokedex Atlas

## Navegacion
- Tabs principales: Pokedex, Detalle, Equipo.
- Detalle se habilita cuando existe seleccion.

## Pantallas
### Pokedex
- Hero con titulo, descripcion y tabs.
- Input de busqueda.
- Grid de tarjetas con imagen, ID y nombre.

### Detalle
- Imagen grande y datos principales.
- Chips con tipos.
- Boton para agregar al equipo y volver a lista.
- Tarjetas con info rapida y stats.

### Equipo
- Lista de favoritos guardados.
- Boton para limpiar equipo.
- Mensaje vacio si no hay items.

## Estados
- Loading list: mensaje simple.
- Loading detail: mensaje simple.
- Error: banner rojo suave.

## Copys
- Hero: "Pokedex Atlas" y descripcion corta.
- CTA: "Agregar al equipo", "Volver a lista".
