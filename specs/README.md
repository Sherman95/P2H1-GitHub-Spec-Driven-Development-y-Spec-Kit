# Spec Kit - P2H1

Este folder contiene la especificacion del producto y el plan de trabajo.

## Archivos
- 01-product-spec.md: objetivos, historias, alcance
- 02-ux-spec.md: estructura de pantallas y estados
- 03-technical-spec.md: arquitectura, datos y API
- 04-task-plan.md: tareas concretas para implementar

## Flujo sugerido

3. Confirmar tecnicos en 03-technical-spec.md
4. Ejecutar tareas en 04-task-plan.md
